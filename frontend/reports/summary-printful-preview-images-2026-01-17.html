Q
Why does Printful provide multiple preview images?
Different preview images may show the product from various angles, with different lighting, or in different contexts (e.g., lifestyle shots, close-ups, etc.).
Q
Which preview image should I use as template?
This depends on your use case. The primary mockup_url is typically the standard product view. Additional previews in extra array may provide alternative angles or styles.
Q
How can I see all available preview images for a specific variant?
Use debugMockupResponse(variantId) function in lib/actions/printful.ts or run debug_gen_diag.ts script to inspect the full mockup response.
Q
Will changing the template affect existing mockups?
Changing which preview image is used as a template will only affect new mockup generations. Existing stored mockup URLs will remain unchanged.
